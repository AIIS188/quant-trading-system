## 🔄 系统架构

```
quant-trading/
├── main.py                    # 主程序
├── src/
│   ├── data_fetcher.py        # 数据获取 (akshare + 缓存)
│   ├── data_fetcher_sina.py   # K 线数据 (新浪财经) ✅
│   ├── data_fetcher_tushare.py# Tushare 接口 (权限不足)
│   ├── data_fetcher_unified.py# 统一接口 (自动降级)
│   ├── hotspot_analyzer.py    # 热点分析
│   ├── strategy_engine.py     # 策略引擎 (需要 K 线) ✅
│   ├── strategy_simple.py     # 简化策略 (无需 K 线)
│   ├── risk_controller.py     # 风控系统
│   ├── backtest.py            # 回测框架
│   └── daily_review.py        # 每日复盘
├── config/
│   ├── config.json            # 系统配置
│   └── tushare_token.txt      # Tushare Token
├── data/cache/                # 数据缓存
└── logs/                      # 日志报告
```
