## 🚀 方案 B: 聚宽模拟盘 (推荐)

### 1. 注册聚宽账号

**步骤**:

1. **访问官网**
   ```
   https://www.joinquant.com/
   ```

2. **注册账号**
   ```
   点击右上角"注册"
   输入手机号
   获取验证码
   设置密码
   ```

3. **实名认证**
   ```
   登录后 → 个人中心 → 实名认证
   上传身份证照片
   等待审核 (通常 10 分钟)
   ```

4. **开通量化交易**
   ```
   个人中心 → 量化交易 → 开通权限
   阅读风险告知书
   确认开通
   ```

### 2. 获取 API 凭证

1. **进入 API 管理**
   ```
   个人中心 → API 密钥
   ```

2. **创建密钥**
   ```
   点击"创建密钥"
   记录：
   - 用户名 (手机号)
   - 密码 (自己设置)
   ```

3. **保存凭证**
   ```
   ⚠️ 重要：将用户名和密码保存到安全位置
   建议：创建配置文件 config/joinquant_config.txt
   ```

### 3. 安装 SDK


```python
# 早盘突破策略
def check_morning_breakout(self, symbol: str):
    # 高开幅度 2-5%
    if not (2 <= gap_up <= 5):
        return None
    
    # 量比 > 1.5
    if volume_ratio < 1.5:
        return None
    
    # 仓位 10%
    quantity = int(self.capital * 0.1 / current_price / 100) * 100

# 盘中回调策略
def check_pullback_buy(self, symbol: str):
    # 回调 3-5%
    if not (3 <= pullback <= 5):
        return None
    
    # 量比 < 0.8 (缩量)
    if volume_ratio > 0.8:
        return None

# 尾盘买入策略
def check_tail_buy(self, symbol: str):
    # 涨幅 3-7%
    if not (3 <= change_pct <= 7):
        return None
```

- 单笔仓位≤10%

- 定期备份交易记录

## 📞 常见问题

### Q1: 自建模拟盘和聚宽有什么区别？

A: 
- **自建**: 本地记录，快速测试，无需注册
- **聚宽**: 真实环境，完整记录，可转实盘

**建议**: 先用自建测试，稳定后切换聚宽

### Q2: 如何修改监控股票？

A: 编辑 `auto_trading.py` 中的 `watch_list` 列表

### Q3: 如何调整策略参数？

A: 编辑 `src/intraday_strategy.py` 中的策略条件

### Q4: 聚宽认证失败怎么办？

A: 
1. 检查账号密码是否正确
2. 确认已开通量化交易权限
3. 检查网络连接
4. 联系聚宽客服

### Q5: 如何查看历史交易？

A: 
```bash
# 自建模拟盘
cat quant-trading/data/paper/paper_state.json

# 聚宽
python -c "from joinquant_integration import JoinQuantTrader; j = JoinQuantTrader(); print(j.get_order_history(days=7))"
```
