## 🤖 我是如何工作的

### 一句话总结

> **我是一个自动化 A 股量化交易助手，每日收盘后自动扫描市场，通过 5 大策略分析 62+ 涨停股，结合情绪得分和资金流数据，生成交易信号和操作建议。**

### 2. 情绪得分计算示例

**假设数据**:
- 涨停家数：62 只
- 跌停家数：0 只
- 最高连板：3 连板
- 净流入行业占比：76%
- 北向净流入：0 亿

**计算过程**:
```python
涨停得分 = min(62/100 * 100, 100) = 62
跌停得分 = min((100-0)/100 * 100, 100) = 100
连板得分 = min(3 * 20, 100) = 60
资金流得分 = 76
北向得分 = 50 (无净流入，基础分)

情绪得分 = 62×0.30 + 100×0.20 + 60×0.20 + 76×0.15 + 50×0.15
         = 18.6 + 20 + 12 + 11.4 + 7.5
         = 69.5 分

→ 上升期 (积极做多)
```

**实际今日 (2026-03-06) 得分：20 分 (退潮期)**

## 🛡️ 四、风控机制

### 1. 仓位控制

```python
def calculate_position(price, confidence):
    """
    根据情绪和置信度计算仓位
    """
    base_position = total_capital * 0.10  # 基础 10%
    
    # 根据情绪调整
    if emotion_score < 30:
        max_position = 0  # 空仓
    elif emotion_score < 50:
        max_position = 0.30  # 轻仓
    elif emotion_score < 70:
        max_position = 0.70  # 积极
    else:
        max_position = 0.50  # 谨慎
    
    # 根据置信度调整
    position = base_position * confidence
    
    # 不超过上限
    position = min(position, max_position)
    
    return position
```

### 2. 止损止盈

```python
def set_stop_profit(signal):
    """
    设置止损止盈位
    """
    if signal.strategy == "龙头回调":
        signal.stop_loss = signal.price * 0.95  # -5%
        signal.take_profit = signal.price * 1.15  # +15%
    
    elif signal.strategy == "突破":
        signal.stop_loss = signal.price * 0.92  # -8%
        signal.take_profit = signal.price * 1.20  # +20%
    
    elif signal.strategy == "龙头接力":
        signal.stop_loss = signal.price * 0.90  # 断板止损
        signal.take_profit = None  # 开板止盈
    
    return signal
```
