# 量化系统 - 自主运行清单

## 每日必做

### 15:30 收盘后扫描
- [ ] 获取涨停板数据
- [ ] 获取龙虎榜数据
- [ ] 计算情绪得分
- [ ] 生成交易信号
- [ ] 风控检查
- [ ] 保存扫描日志

### 20:00 复盘
- [ ] 统计今日交易
- [ ] 分析盈亏原因
- [ ] 更新策略胜率
- [ ] 生成复盘报告

### 实时监控
- [ ] 情绪得分 > 60 → 准备做多
- [ ] 情绪得分 < 30 → 空仓观望
- [ ] 连续亏损 3 次 → 暂停交易
- [ ] 回撤 > 8% → 停止交易

## 每周必做

### 周日
- [ ] 策略胜率统计
- [ ] 淘汰胜率 < 50% 策略
- [ ] 强化胜率 > 60% 策略
- [ ] 生成周度报告

## 决策阈值

### 买入
- 情绪得分 ≥ 60
- 策略信号生成
- 风控检查通过
- 单笔 ≤ 10%

### 卖出
- 达到目标收益 (+10%)
- 触及止损线 (-5%)
- 情绪转弱 (< 40)
- 策略卖出信号

### 停止交易
- 连续亏损 3 次
- 回撤 > 8%
- 单日亏损 > 3%
