> 基于实时 K 线监控的 A 股短线交易策略系统

**作者**: AIIS188  
**版本**: v2.0  
**最后更新**: 2026-03-07

---

## 🎯 系统特点

- ✅ **实时 K 线监控** - 1-3 秒延迟
- ✅ **盘中自动交易** - 捕捉最佳买点
- ✅ **三大策略** - 早盘突破/盘中回调/尾盘买入
- ✅ **严格风控** - 止损止盈/仓位控制
- ✅ **T+1 优化** - 专为 A 股设计
- ✅ **自动运行** - 定时任务/自动报告

---

## 📊 策略说明

### 1. 早盘突破 (09:30-10:30)
- **条件**: 高开 2-5% + 量比>1.5
- **止损**: -5%
- **止盈**: +15%
- **仓位**: 10%

### 2. 盘中回调 (10:30-14:30)
- **条件**: 回调 3-5% + 缩量
- **止损**: -5%
- **止盈**: +10%
- **仓位**: 10%

### 3. 尾盘买入 (14:30-15:00) ⭐
- **条件**: 涨幅 3-7% + 尾盘稳定
- **止损**: -5%
- **止盈**: +10%
- **仓位**: 10%

## 🚀 快速开始

### 安装依赖
```bash
pip install akshare pandas requests numpy
```

### 配置监控列表
编辑 `auto_trading.py`:
```python
watch_list = [
    '000533',  # 顺钠股份
    '605268',  # 王力安防
    '600545',  # 卓郎智能
    # ... 添加更多股票
]
```

### 启动自动交易
```bash
# 测试模式
python auto_trading.py --test

# 运行一次
python auto_trading.py --once

# 连续监控 (交易时间)
python auto_trading.py --interval 300
```

### 设置定时任务
```bash
# 编辑 crontab
crontab -e

# 添加任务 (交易日自动运行)
20 9 * * 1-5 cd /path/to/quant-trading && python auto_trading.py --once
30 14 * * 1-5 cd /path/to/quant-trading && python auto_trading.py --once
50 14 * * 1-5 cd /path/to/quant-trading && python auto_trading.py --tail
```

## 🛡️ 风控系统

- 单笔仓位 ≤ 10%
- 单日交易 ≤ 5 笔
- 单笔止损 -5%
- 单日亏损 ≤ 3%
- 最大回撤 ≤ 8%
- 连续亏损 ≤ 3 次 → 暂停 1 天
